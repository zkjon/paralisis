
# 🧠 parálisis del sueño

> cuando la mente despierta, pero el cuerpo no responde

un proyecto educativo y narrativo que explora la parálisis del sueño: qué es, por qué ocurre y cómo se experimenta este fenómeno entre el sueño y la realidad.

---

## 🌙 sobre el proyecto

la parálisis del sueño es una experiencia en la que una persona está consciente, pero no puede moverse ni hablar.  
este proyecto busca explicar este fenómeno desde una doble perspectiva:

- 🔬 científica → neurología, fases del sueño, causas  
- 🌑 experiencial → sensaciones, miedo, percepción  

el objetivo es hacer comprensible algo que muchas veces se percibe como inexplicable.

---

## 🧩 contenido

el proyecto está estructurado como una documentación:

- **qué es la parálisis del sueño**
- **por qué ocurre**
- **cómo funciona el sueño rem**
- **tipos de experiencias**
- **alucinaciones**
- **cómo prevenirla**
- **qué hacer durante un episodio**

* violeta / índigo

* contrastes suaves
